# 🗺️ Product Roadmap

## Phase 1: MVP Foundation (Current)
**Goal:** Enable secure visitor tracking for a single hospital chain with multiple branches.
**Timeline:** Q1 2025

### Milestones
- [x] **M1: Core Infrastructure**
  - Monorepo setup (NestJS + Next.js).
  - Database Schema (Prisma + MySQL).
  - Auth System (OTP + JWT).
- [ ] **M2: Administration Module**
  - Super Admin: Manage Chains & Branches.
  - Chain/Branch Admin: Manage Users (Staff/Security).
- [ ] **M3: Visitor Operations**
  - Security: Check-in/Check-out flows.
  - Visitor: QR Code registration.
  - Staff: Approval workflow.

## Phase 2: Enhanced Experience & Scale
**Goal:** Improve UX and add analytics.
**Timeline:** Q1 2026

### Features
- **Analytics Dashboard:** Visual reports for visitor traffic.
- **WhatsApp Integration:** Send gate passes via WhatsApp.
- **Pre-booking:** Allow visitors to schedule visits in advance.
- **Multi-tenant SaaS:** Support completely independent organizations.

## Phase 3: Healthcare Community Platform (Long-term)
**Goal:** Leverage verified identities for professional networking.
**Timeline:** 2026+

### Features
- **Professional Profiles:** Verified doctors/staff profiles.
- **Knowledge Sharing:** Medical forums and case discussions.
